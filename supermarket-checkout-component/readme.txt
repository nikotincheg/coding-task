Start application instruction:
1. Build the app using following maven command: mvn clean install
2. Go to the 'target' directory
3. Run the app using following command (tested using Windows): java -jar supermarket-checkout-component.jar

Additional information:
1. Java 8 is used
2. Used maven version - 3.3.9
3. Lombok plugin is required to correct work with the project in IntelliJ IDEA
4. Only several classes were covered by tests - just for example
